# Installation
 1. Clone the repository in a new directory (Or download the repository)
 2. Put all contents of the cloned directory in your minetest root folder
 3. Put all core mods (default,bucket etc...) into your "BlockForge2/mods" game folder
 4. FUN FUN FUN FUN

# Warning
This mods overwrite some core mods and builin mods.If you want to deactivate the corehack,put "lulzpack_deactivate_corehack = true" in your minetest.conf (you can find this file in the root folder) and don't install the "builtin" folder in this repository.
